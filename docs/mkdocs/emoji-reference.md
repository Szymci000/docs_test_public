## Użycie

Emoji można używać na dwa sposoby:

1. **Krótkie kody** - używając składni `:kod_emoji:`
2. **Unicode** - bezpośrednio wpisując emoji

Przykład:
````markdown
:smile: :heart: :rocket: :thumbsup:
````

## Wskazówki

- Używaj emoji oszczędnie - zbyt dużo może rozpraszać
- Emoji są przydatne do szybkiego przekazywania emocji lub statusu
- Nie używaj emoji w formalnych dokumentach technicznych
- Emoji są automatycznie konwertowane na ikony SVG dla lepszej wydajności
- Możesz używać emoji w tytułach, listach i innych elementach markdown
- Kopiowanie: Kliknij na kod w tabeli, aby skopiować
- Wyszukiwanie: Użyj Ctrl+F aby znaleźć konkretne emoji
