# Listy definicji

Listy definicji pozwalają na tworzenie list terminów z ich definicjami. To przydatne do tworzenia słowników, glosariuszy lub wyjaśniania terminów.

## Użycie

Listy definicji tworzy się używając składni z dwukropkiem:

````markdown
Termin 1
:   Definicja terminu 1. Może zawierać wiele linii.

Termin 2
:   Definicja terminu 2.
:   Można również dodać kolejną linię definicji.

Termin 3
:   Definicja terminu 3 z **formatowaniem**.
````

## Przykłady

### Podstawowa lista definicji

HTML
:   HyperText Markup Language - język znaczników używany do tworzenia stron internetowych.

CSS
:   Cascading Style Sheets - język służący do stylowania dokumentów HTML.

JavaScript
:   Język programowania używany do tworzenia interaktywnych stron internetowych.

### Lista definicji z formatowaniem

API
:   Application Programming Interface - interfejs programistyczny aplikacji pozwalający na komunikację między różnymi systemami.

REST
:   Representational State Transfer - architektura sieciowa używana do projektowania interfejsów API.

JSON
:   JavaScript Object Notation - lekki format wymiany danych używany do przesyłania danych między aplikacjami.

### Lista definicji z wieloma liniami

MVC
:   Model-View-Controller - wzorzec architektoniczny używany w projektowaniu aplikacji.
:   Model reprezentuje dane, View reprezentuje interfejs użytkownika, a Controller zarządza logiką.

CRUD
:   Create, Read, Update, Delete - podstawowe operacje wykonywane na danych.
:   Te operacje są fundamentalne dla większości aplikacji.

### Lista definicji z linkami

MkDocs
:   Generator dokumentacji statycznej dla projektów. Więcej informacji na [stronie projektu](https://www.mkdocs.org/).

Material for MkDocs
:   Motyw dla MkDocs oparty na Material Design. Więcej informacji w [dokumentacji](https://squidfunk.github.io/mkdocs-material/).

### Lista definicji z kodami

CLI
:   Command Line Interface - interfejs wiersza poleceń.
:   Przykład użycia: `npm install package-name`

IDE
:   Integrated Development Environment - zintegrowane środowisko programistyczne.
:   Popularne IDE: Visual Studio Code, IntelliJ IDEA, Eclipse.

### Lista definicji w kontekście

OAuth
:   Open Authorization - protokół autoryzacji używany do bezpiecznego udostępniania zasobów.

JWT
:   JSON Web Token - standard otwarty używany do bezpiecznego przesyłania informacji między stronami.

### Lista definicji z listami

HTTP
:   HyperText Transfer Protocol - protokół komunikacyjny używany do przesyłania danych w sieci.
:   Metody HTTP:
:   - GET - pobieranie danych
:   - POST - wysyłanie danych
:   - PUT - aktualizacja danych
:   - DELETE - usuwanie danych

## Wskazówki

- Używaj list definicji do tworzenia glosariuszy i słowników
- Definicje mogą zawierać wiele linii
- Możesz używać formatowania markdown w definicjach
- Listy definicji są przydatne do wyjaśniania terminów technicznych
- Każda definicja musi zaczynać się od dwukropka i być wcięta
